# CRAFT Framework

**Best for:** General-purpose prompt writing, everyday tasks where you need clarity without heavy structure.

## Template Structure

Rewrite the prompt into these sections:

### Context
Provide the background information and situation. What is the project, problem, or scenario? What has already been done or decided?

### Role
Define who the AI should be — the expertise or perspective to adopt. Match this to the domain of the task.

### Action
State the specific task using a precise verb. What exactly should be done? Be concrete and unambiguous.

### Format
Specify the desired output format — code block, bullet list, table, JSON, prose paragraph, step-by-step guide, etc.

### Tone
Set the communication style — technical, conversational, formal, concise, detailed, tutorial-style, etc. Match this to the intended audience and purpose.

## Rules for This Framework
- Every section should be brief — CRAFT is meant to be lightweight
- Not every section needs extensive detail — use what the original prompt provides
- Action is the most important section — make it crystal clear
