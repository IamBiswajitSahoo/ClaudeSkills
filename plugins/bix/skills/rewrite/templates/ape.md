# APE Framework

**Best for:** Quick tasks, simple requests, one-off prompts where minimal structure is sufficient.

## Template Structure

Rewrite the prompt into these sections:

### Action
State what needs to be done — use a clear, direct verb phrase. One sentence is ideal.

### Purpose
Explain why this is needed — the motivation, use case, or problem being solved. This helps the AI prioritize and make judgment calls.

### Expectation
Describe what good output looks like — format, quality bar, length, or specific characteristics of the desired result.

## Rules for This Framework
- This framework is intentionally minimal — do not over-elaborate
- Three concise sentences can be sufficient for a well-structured APE prompt
- Use this when the task is straightforward and doesn't need multi-step decomposition
